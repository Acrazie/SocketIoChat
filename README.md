## Getting Started

Suivez le guide d'installation et tout ira bien :)

### Prerequisites

```sh
sudo npm cache clean -f
sudo npm install -g n
sudo n stable
```

### Installation

1. Clone the repo
   ```sh
   git@github.com:Acrazie/SocketIoChat.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the project
   ```sh
   node index.js
   ```
