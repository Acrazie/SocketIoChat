# Chat Application

This is a real-time chat application built using Socket.IO and Express.js.

## Features

- Real-time messaging
- User connection and disconnection notifications
- Room-based chat

## Technologies Used

- Node.js
- Express.js
- Socket.IO
- HTML/CSS/JavaScript
